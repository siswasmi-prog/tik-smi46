1. Anatomi dan Struktur Dasar HTML

Setiap halaman web memiliki anatomi dasar yang terdiri dari tag-tag wajib yang membentuk struktur utamanya. Memahami anatomi ini sangat penting sebagai fondasi untuk membangun halaman web.

  • <!DOCTYPE html>: Ini bukan tag HTML, melainkan sebuah deklarasi yang memberitahu browser bahwa dokumen ini adalah dokumen HTML5. Deklarasi ini harus selalu diletakkan di baris pertama kode.
  • <html>: Ini adalah elemen akar yang membungkus seluruh konten di halaman HTML. Semua elemen HTML lainnya, kecuali deklarasi DOCTYPE, harus berada di dalam tag ini.
  • <head>: Bagian ini digunakan untuk menyimpan metadata tentang halaman, seperti judul halaman, deskripsi, atau link ke file CSS dan JavaScript. Konten di dalam tag ini tidak akan terlihat oleh pengguna di browser.
  • <body>: Bagian utama di mana semua konten yang terlihat oleh pengguna, seperti teks, gambar, video, dan link, ditempatkan.

Contoh Penggunaan:

Untuk membuat struktur html otomatis di VS Code, ketik html dan pilih html5. Berikut adalah contoh struktur dasar HTML yang benar dan lengkap:

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Halaman Pertamaku</title>
</head>
<body>
    <h1>Selamat Datang di Webku!</h1>
    <p>Ini adalah paragraf pertama di halaman web yang saya buat.</p>
</body>
</html>

Soal Latihan:

1. Apa fungsi dari tag <head> dalam struktur HTML?

2. Tuliskan struktur dasar HTML lengkap dengan semua tag wajibnya.

2. Konten H1, Paragraf, dan List

Heading (`<h1>` hingga `<h6>`)

Tag heading digunakan untuk membuat judul dan subjudul. `<h1>` adalah yang paling penting (digunakan untuk judul utama), dan `<h6>` adalah yang paling tidak penting. Penggunaannya membantu struktur halaman dan penting untuk SEO.

Paragraf (`<p>`)

Tag `<p>` digunakan untuk menyusun blok teks. Browser secara otomatis menambahkan margin di atas dan bawah tag ini untuk membedakannya dari elemen lain.

List (`<ul>` dan `<ol>`)

Daftar digunakan untuk mengorganisir informasi.

  • `<ul>` (unordered list) digunakan untuk daftar yang tidak memiliki urutan, ditandai dengan bullet point.
  • `<ol>` (ordered list) digunakan untuk daftar yang berurutan (misalnya, langkah-langkah dalam resep), ditandai dengan angka atau huruf.

Contoh Penggunaan:

<h1>Resep Kue Cokelat</h1>
<p>Berikut adalah bahan-bahan yang Anda perlukan:</p>
<ul>
    <li>Tepung terigu</li>
    <li>Cokelat bubuk</li>
    <li>Telur</li>
</ul>
<h2>Langkah-langkah Membuatnya</h2>
<ol>
    <li>Campurkan semua bahan kering.</li>
    <li>Tambahkan telur dan aduk hingga merata.</li>
    <li>Panggang di oven selama 30 menit.</li>
</ol>

Soal Latihan:

1. Bagaimana cara membuat judul utama dengan HTML?

2. Buatlah daftar berurutan untuk resep kopi, minimal 3 langkah.

3. Gambar dan Video

Gambar (`<img>`)

Tag <img> digunakan untuk menyematkan gambar. Tag ini tidak memiliki tag penutup. Atribut terpenting adalah `src` (sumber gambar) dan `alt` (teks alternatif).

  • `src`: Menentukan path ke file gambar.
  • `alt`: Memberikan deskripsi teks untuk gambar. Ini sangat penting untuk aksesibilitas (untuk pembaca layar) dan SEO (jika gambar tidak dapat dimuat).

Video (`<video>`)

Tag `<video>` digunakan untuk menyematkan video. Anda dapat menyertakan atribut seperti `width`, `height`, dan `controls` untuk memanipulasi tampilannya.

  • `src`: Menentukan path ke file video.
  • `controls`: Menambahkan kontrol pemutaran seperti play/pause dan volume.
  • `autoplay`: Memulai pemutaran video secara otomatis saat halaman dimuat.

Contoh Penggunaan:

<img src="https://placehold.co/600x400/000000/FFFFFF?text=Contoh+Gambar" alt="Deskripsi gambar contoh">

<video src="belajar.mp4" width="640" height="360" controls>
    Browser Anda tidak mendukung tag video.
</video>

Soal Latihan:

1. Jelaskan fungsi dari atribut `alt` pada tag `<img>`.

2. Buatlah tag video yang menyematkan file `belajar.mp4` dengan lebar 800px dan kontrol.

4. Tabel (`<table>`)

Tabel digunakan untuk menampilkan data dalam format baris dan kolom. Struktur tabel yang semantik terdiri dari beberapa elemen utama:

  • `<table>`: Elemen utama yang membungkus seluruh tabel.
  • `<thead>`: Mengelompokkan baris header (judul kolom).
  • `<tbody>`: Mengelompokkan baris data utama.
  • `<tr>`: Mendefinisikan baris dalam tabel.
  • `<th>`: Mendefinisikan sel header, biasanya diletakkan di dalam `<thead>`.
  • `<td>`: Mendefinisikan sel data.

Contoh Penggunaan:

<table>
    <thead>
        <tr>
            <th>Nama Produk</th>
            <th>Harga</th>
            <th>Stok</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Laptop</td>
            <td>$1200</td>
            <td>50</td>
        </tr>
        <tr>
            <td>Mouse</td>
            <td>$25</td>
            <td>200</td>
        </tr>
    </tbody>
</table>

Soal Latihan:

1. Bagaimana cara membuat baris baru dalam tabel HTML?

2. Buatlah tabel dengan dua baris data dan tiga kolom: "Nama Produk," "Harga," dan "Stok."

5. Semantic HTML, `<div>`, `<span>`, Atribut `class` dan `id`

Semantic HTML

Tag semantik memberikan makna pada kontennya. Contoh: `<header>` untuk bagian kepala, `<nav>` untuk navigasi, `<main>` untuk konten utama, dan `<footer>` untuk bagian bawah halaman. Penggunaan tag ini penting untuk aksesibilitas dan SEO.

`<div>` dan `<span>`

Tag `<div>` (block-level) dan `<span>` (inline) adalah tag non-semantik. Mereka tidak memberikan makna khusus, melainkan berfungsi sebagai wadah untuk mengelompokkan elemen lain, biasanya untuk keperluan styling atau JavaScript.

Atribut `class` dan `id`

Atribut `class` dan `id` digunakan sebagai pengenal untuk menargetkan elemen dengan CSS atau JavaScript. **`class`** dapat digunakan oleh beberapa elemen, sementara **`id`** harus unik untuk setiap elemen.

Contoh Penggunaan:

<header>
    <nav>
        <a href="/">Beranda</a>
    </nav>
</header>

<div class="kontainer-utama">
    <p>Selamat datang, <span class="nama-pengguna" id="user-name">Pengguna</span>!</p>
</div>

<footer>
    <p>© 2024 Aplikasi E-learning</p>
</footer>

Soal Latihan:

1. Apa perbedaan utama antara `<div>` dan `<section>` dari segi semantik?

2. Mengapa penting untuk menggunakan atribut `class` dan `id`?

3. Buatlah sebuah elemen `<div>` dengan `id` "profil-pengguna" dan di dalamnya terdapat `<p>` dengan `class` "nama-lengkap".

1. Selektor CSS, Properti Warna dan Teks

CSS (Cascading Style Sheets) digunakan untuk menata tampilan halaman web. Dasar dari CSS adalah **selektor** dan **properti**.

Selektor

Selektor adalah pola yang digunakan untuk "menemukan" dan menargetkan elemen HTML yang ingin Anda beri gaya. Ada beberapa jenis selektor:

  • Selektor Elemen: Memilih semua elemen dari jenis tertentu, contohnya `h1 { ... }`.
  • Selektor Class: Memilih semua elemen dengan atribut `class` tertentu, contohnya `.kontainer { ... }`.
  • Selektor ID: Memilih elemen dengan atribut `id` yang unik, contohnya `#header { ... }`.

Properti Warna dan Teks

Properti-properti ini digunakan untuk mengubah tampilan teks dan warna.

  • `color`: Mengubah warna teks.
  • `background-color`: Mengubah warna latar belakang.
  • `font-size`: Mengubah ukuran font.
  • `font-family`: Mengubah jenis font.

Contoh Penggunaan:

/* Menggunakan selektor elemen */
p {
    color: #fca5a5; /* Warna teks merah muda */
    font-size: 1.25rem; /* Ukuran font lebih besar */
}

/* Menggunakan selektor class */
.highlight {
    background-color: #fde047; /* Latar belakang kuning */
    font-weight: bold; /* Teks tebal */
}

/* Menggunakan selektor ID */
#judul-utama {
    color: #60a5fa; /* Warna teks biru */
    text-align: center; /* Teks di tengah */
}

Soal Latihan:

1. Bagaimana cara memilih semua paragraf dan memberikan warna teks merah?

2. Buatlah selektor yang memilih elemen dengan `class` `highlight` dan berikan `background-color` kuning.

2. Box Model

Setiap elemen HTML dianggap sebagai kotak, yang terdiri dari empat bagian utama: konten, padding, border, dan margin. Memahami model ini sangat penting untuk mengatur tata letak dan jarak antar elemen.

  • Content (Konten): Area di tengah kotak, tempat teks atau gambar berada.
  • Padding: Ruang kosong di antara konten dan border. Padding berfungsi untuk memberikan "ruang bernapas" di dalam kotak.
  • Border: Garis yang mengelilingi padding dan konten. Border dapat diatur ketebalan, gaya, dan warnanya.
  • Margin: Ruang kosong di luar border, digunakan untuk memisahkan satu elemen dari elemen lainnya.

Contoh Penggunaan:

.kontainer {
    background-color: #3b82f6;
    padding: 20px; /* Ruang di dalam kotak, antara konten dan border */
    border: 5px solid #2563eb; /* Garis pembatas kotak */
    margin: 30px; /* Ruang di luar kotak, memisahkan dari elemen lain */
}

Soal Latihan:

1. Jelaskan perbedaan antara padding dan margin.

2. Buatlah sebuah `class` bernama `kartu` yang memiliki padding 15px, border solid abu-abu, dan margin 20px.

3. Display, Posisi, Flexbox, dan Grid

Properti `display` mengontrol bagaimana sebuah elemen ditampilkan. Nilai-nilai umum yang digunakan adalah `block`, `inline`, dan `flex`/`grid` untuk tata letak modern.

Flexbox (Layout Modern)

**Flexbox** (Flexible Box) adalah model tata letak satu dimensi untuk mendistribusikan ruang dan menyelaraskan item dalam sebuah kontainer. Ini sangat berguna untuk menempatkan item dalam satu baris atau satu kolom.

Grid Layout

**Grid Layout** adalah sistem tata letak dua dimensi yang dirancang untuk mengelola tata letak baris dan kolom secara bersamaan. Ini ideal untuk membuat tata letak halaman yang kompleks.

Contoh Penggunaan:

/* Flexbox untuk menempatkan item di tengah */
.flex-container {
    display: flex;
    justify-content: center; /* Menyusun item secara horizontal di tengah */
    align-items: center; /* Menyusun item secara vertikal di tengah */
    height: 200px;
    background-color: #4a5568;
}

/* Grid untuk membuat tata letak 3 kolom */
.grid-container {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr; /* Membagi ruang menjadi 3 kolom yang sama */
    gap: 16px; /* Jarak antar kolom dan baris */
}

Soal Latihan:

1. Kapan Anda harus menggunakan Flexbox, dan kapan Grid?

2. Buatlah tata letak Flexbox yang menempatkan tiga item di tengah secara horizontal dan vertikal.

4. Responsif, Transisi, dan Animasi

Responsif (`@media` Queries)

Desain web responsif adalah praktik membuat website yang beradaptasi dengan ukuran layar perangkat yang berbeda. `@media` queries memungkinkan Anda menerapkan gaya CSS khusus berdasarkan kondisi, seperti lebar layar.

Transisi dan Animasi

**Transisi** digunakan untuk membuat perubahan properti CSS (misalnya, warna atau ukuran) menjadi lebih halus. **Animasi** memungkinkan Anda membuat pergerakan yang lebih kompleks dan dapat diatur melalui keyframes.

Contoh Penggunaan:

/* Media Query untuk layar kecil (mobile) */
@media screen and (max-width: 768px) {
    body {
        background-color: #1a202c; /* Warna latar belakang gelap untuk mobile */
    }
}

/* Transisi halus saat tombol di-hover */
.tombol-hover {
    background-color: #3b82f6;
    transition: background-color 0.3s ease;
}

.tombol-hover:hover {
    background-color: #2563eb;
}

Soal Latihan:

1. Apa yang dimaksud dengan desain web responsif?

2. Buatlah aturan `@media` yang mengubah warna latar belakang `<body>` menjadi merah saat lebar layar di bawah 768px.

5. Pseudo-Class, Pseudo-Element, dan Custom Properties

Pseudo-Class

Pseudo-class digunakan untuk menyeleksi elemen berdasarkan statusnya. Contohnya, `:hover` saat kursor berada di atas elemen, atau `:first-child` untuk elemen anak pertama.

Custom Properties (CSS Variables)

Variabel CSS, atau Custom Properties, memungkinkan Anda untuk menyimpan nilai dalam satu tempat dan menggunakannya kembali di seluruh stylesheet. Ini membuat kode lebih mudah diatur dan diubah.

Contoh Penggunaan:

/* Pseudo-class untuk link */
a:hover {
    text-decoration: underline;
    color: #60a5fa;
}

/* Custom Properties (Variabel CSS) */
:root {
    --warna-utama: #10b981;
    --lebar-kontainer: 960px;
}

.card {
    background-color: var(--warna-utama);
    max-width: var(--lebar-kontainer);
}

Soal Latihan:

1. Bagaimana cara mengubah warna teks link menjadi biru saat dihover?

2. Buatlah sebuah Custom Property bernama `--lebar-konten` dan gunakan di salah satu selektor.
